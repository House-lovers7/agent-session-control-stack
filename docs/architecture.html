Agent Session Control Stack Architecture

ASCS separates long-running AI coding agent stability into four layer contracts, runtime bindings, and a conservative evidence measurement path.

README Markdown architecture Claim-boundary model
This HTML view is explanatory. It does not claim that ASCS improves productivity, validates model quality, or proves the full-stack composition effect. Experiment 004 currently demonstrates read-only claim-boundary classification for recorded evidence.
1. Compression Reduce bulky context. Keep byte-exact values as text.
2. Health Detection Detect hot sessions and keep compact decisions single-owner.
3. Checkpointing Preserve plan, decisions, failed attempts, and topology.
4. Recovery Resume from state. Treat summaries as hypotheses.
